# Animation Styles

YongUI의 애니메이션 시스템은 일관되고 자연스러운 모션을 제공합니다.
성능 최적화와 접근성을 고려한 6가지 핵심 애니메이션 스타일을 제공합니다.

## 애니메이션 스타일

## 사용 가이드

### 기본 사용법

```tsx
// 단일 애니메이션
<div className={css({ animationStyle: 'spin' })}>
  로딩 중...
</div>

// 조건부 애니메이션
<div className={css({
  animationStyle: isLoading ? 'pulse' : undefined
})}>
  상태 표시
</div>
```

### 커스텀 타이밍

```tsx
// 애니메이션 속도 조절
<div
  className={css({
    animationStyle: 'fadeIn',
    animationDuration: '0.5s',
  })}
>
  느린 페이드
</div>;

// 순차적 애니메이션
{
  items.map((item, index) => (
    <li
      className={css({
        animationStyle: 'fadeIn',
        animationDelay: `${index * 0.1}s`,
      })}
    >
      {item}
    </li>
  ));
}
```

## 성능 최적화

### GPU 가속

```tsx
// will-change 사용
<div
  className={css({
    willChange: 'transform',
    animationStyle: 'scaleUp',
  })}
>
  최적화된 애니메이션
</div>
```

### 접근성

```tsx
// 모션 설정 존중
<div className={css({
  animationStyle: 'bounce',
  '@media (prefers-reduced-motion: reduce)': {
    animationStyle: 'none'
  }
})}>
  접근성 고려
</div>

// ARIA 속성
<div
  className={css({ animationStyle: 'spin' })}
  role="status"
  aria-live="polite"
  aria-label="로딩 중"
>
  <Spinner />
</div>
```

## 실제 구현 예시

### 로딩 버튼

```tsx
const LoadingButton = ({ loading, children, ...props }) => (
  <Button {...props}>
    {loading ? (
      <span className={css({ animationStyle: 'spin' })}>
        <icons.loader size={16} />
      </span>
    ) : (
      children
    )}
  </Button>
);
```

### 알림 뱃지

```tsx
const NotificationBadge = ({ count, hasNew }) => (
  <div className={css({ position: 'relative' })}>
    <icons.bell size={24} />
    {hasNew && (
      <span
        className={css({
          animationStyle: 'pulse',
          position: 'absolute',
          top: '-4px',
          right: '-4px',
          width: '8px',
          height: '8px',
          borderRadius: '50%',
          bg: 'bgSolid.danger',
        })}
      />
    )}
  </div>
);
```

## 애니메이션 속성

| 스타일           | 지속 시간 | 용도       | 성능 |
| :--------------- | :-------- | :--------- | :--- |
| **spin**         | 1s (반복) | 로딩 표시  | 최적 |
| **pulse**        | 2s (반복) | 주목 유도  | 최적 |
| **bounce**       | 1s (반복) | 강조, 유도 | 양호 |
| **fadeIn/Out**   | 0.3s      | 전환 효과  | 최적 |
| **slideIn/Out**  | 0.3s      | 패널 이동  | 양호 |
| **scaleUp/Down** | 0.2s      | 팝업 효과  | 최적 |

## 모범 사례

### Do

- 목적에 맞는 애니메이션 선택
- 일관된 타이밍 유지
- 성능 모니터링
- 피드백 제공

### Don't

- 과도한 사용
- 0.3초 이상의 긴 전환
- 무의미한 모션
- 강제 애니메이션
