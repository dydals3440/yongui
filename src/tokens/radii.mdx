# Radii (Border Radius)

YongUI의 모서리 둥글기 시스템은 부드럽고 현대적인 인터페이스를 만들기 위한 9단계 스케일을 제공합니다.
일관된 곡률로 시각적 조화를 이루고, 컴포넌트의 성격을 효과적으로 전달합니다.

## 둥글기 원칙

### 🎨 시각적 조화
- **일관된 스케일**: 2px부터 32px까지 체계적인 증가
- **용도별 구분**: 미묘한 둥글기부터 완전한 원형까지
- **컴포넌트 성격**: 둥글기로 친근함과 전문성 균형

### 📐 수학적 체계
- **rem 단위 사용**: 반응형 스케일링 지원
- **픽셀 대응**: 명확한 픽셀 값으로 예측 가능
- **무한 반경**: full 옵션으로 완벽한 원형 구현

### 🎯 의도적 사용
- **기능적 구분**: 버튼, 카드, 입력 필드별 적절한 값
- **브랜드 일관성**: 동일 레벨 요소는 동일한 둥글기
- **시각적 계층**: 둥글기로 요소 간 관계 표현

### xs (2px / 0.125rem)
거의 눈에 띄지 않는 미묘한 둥글기입니다.

```tsx
<div className={css({ borderRadius: 'xs' })}>
  날카로움을 살짝 부드럽게
</div>
```

**사용 사례**
- 작은 체크박스
- 툴팁 말풍선
- 코드 블록
- 미니 태그

**시각적 효과**: 날카로운 모서리를 부드럽게 처리하여 세련된 느낌

### sm (4px / 0.25rem)
섬세하면서도 인지 가능한 둥글기입니다.

```tsx
<button className={css({ borderRadius: 'sm' })}>
  작은 버튼
</button>
```

**사용 사례**
- 작은 버튼
- 인라인 배지
- 셀렉트 박스
- 테이블 셀

**시각적 효과**: 부드러우면서도 전문적인 인상

### md (6px / 0.375rem)
YongUI의 기본 둥글기로 가장 많이 사용됩니다.

```tsx
<input className={css({ borderRadius: 'md' })} />
```

**사용 사례**
- 기본 버튼
- 입력 필드
- 드롭다운 메뉴
- 작은 카드

**시각적 효과**: 친근하면서도 깔끔한 표준 스타일

### lg (8px / 0.5rem)
눈에 띄는 둥글기로 부드러운 인상을 줍니다.

```tsx
<div className={css({
  borderRadius: 'lg',
  p: '16',
  bg: 'bg.neutral'
})}>
  콘텐츠 카드
</div>
```

**사용 사례**
- 카드 컴포넌트
- 모달 창
- 알림 박스
- 섹션 컨테이너

**시각적 효과**: 친근하고 현대적인 느낌

### xl (12px / 0.75rem)
충분히 둥근 모서리로 부드러운 성격을 강조합니다.

```tsx
<div className={css({
  borderRadius: 'xl',
  p: '24'
})}>
  대형 패널
</div>
```

**사용 사례**
- 대형 카드
- 히어로 섹션
- 이미지 컨테이너
- 플로팅 패널

**시각적 효과**: 부드럽고 우아한 인상

### 2xl (16px / 1rem)
매우 둥근 모서리로 강한 부드러움을 표현합니다.

```tsx
<button className={css({
  borderRadius: '2xl',
  px: '32',
  py: '16'
})}>
  큰 액션 버튼
</button>
```

**사용 사례**
- CTA 버튼
- 배너
- 특별 프로모션 카드
- 온보딩 요소

**시각적 효과**: 눈에 띄고 친화적인 느낌

### 3xl (24px / 1.5rem)
거의 타원형에 가까운 둥글기입니다.

```tsx
<div className={css({
  borderRadius: '3xl',
  p: '32'
})}>
  특별 강조 영역
</div>
```

**사용 사례**
- 특수 배너
- 일러스트 컨테이너
- 대화 상자
- 특별 안내

**시각적 효과**: 장난스럽고 캐주얼한 인상

### 4xl (32px / 2rem)
매우 큰 둥글기로 거의 캡슐 형태를 만듭니다.

```tsx
<button className={css({
  borderRadius: '4xl',
  px: '40',
  py: '12'
})}>
  캡슐 버튼
</button>
```

**사용 사례**
- 필 버튼
- 검색 바
- 태그 클라우드
- 스위치 토글

**시각적 효과**: 매우 부드럽고 유기적인 형태

### full (무한)
완벽한 원형을 만들기 위한 특수 값입니다.

```tsx
<div className={css({
  borderRadius: 'full',
  width: '48',
  height: '48'
})}>
  <img src="/avatar.jpg" alt="프로필" />
</div>
```

**사용 사례**
- 아바타 이미지
- 플로팅 액션 버튼
- 상태 인디케이터
- 아이콘 버튼

**시각적 효과**: 완벽한 원형으로 집중도 높음

## 사용 가이드

### 기본 사용법

```tsx
// 전체 모서리 적용
<div className={css({ borderRadius: 'lg' })}>
  모든 모서리가 둥글게
</div>

// 특정 모서리만 적용
<div className={css({
  borderTopLeftRadius: 'lg',
  borderTopRightRadius: 'lg',
  borderBottomRadius: '0'
})}>
  상단만 둥글게
</div>
```

### 방향별 적용

```tsx
// 상단 모서리만
<div className={css({
  borderTopRadius: 'xl' // 상단 좌우
})}>
  상단 둥글기
</div>

// 좌측 모서리만
<div className={css({
  borderLeftRadius: 'lg' // 좌측 상하
})}>
  좌측 둥글기
</div>

// 대각선 모서리
<div className={css({
  borderTopLeftRadius: '2xl',
  borderBottomRightRadius: '2xl'
})}>
  대각선 둥글기
</div>
```

### 반응형 둥글기

```tsx
// 화면 크기별 다른 둥글기
<div className={css({
  borderRadius: ['md', 'lg', 'xl'],
  // 모바일: 6px, 태블릿: 8px, 데스크탑: 12px
})}>
  반응형 카드
</div>
```

## 컴포넌트별 가이드

### 버튼 스타일

```tsx
const ButtonRadius = {
  // 정사각형 아이콘 버튼
  icon: 'md',

  // 일반 버튼
  default: 'md',

  // 큰 CTA 버튼
  cta: '2xl',

  // 필 버튼
  pill: '4xl',

  // 원형 FAB
  fab: 'full'
};

// 사용 예시
<button className={css({
  borderRadius: ButtonRadius.cta,
  px: '32',
  py: '16'
})}>
  시작하기
</button>
```

### 카드 컴포넌트

```tsx
const Card = ({ size = 'default', children }) => {
  const radiusMap = {
    small: 'md',
    default: 'lg',
    large: 'xl',
    hero: '2xl'
  };

  return (
    <div className={css({
      borderRadius: radiusMap[size],
      border: 'neutral',
      p: size === 'small' ? '12' : '24',
      bg: 'bg.neutral'
    })}>
      {children}
    </div>
  );
};
```

### 입력 필드

```tsx
const Input = ({ variant = 'default', ...props }) => {
  const radiusMap = {
    default: 'md',
    search: '4xl',
    textarea: 'lg'
  };

  return (
    <input
      className={css({
        borderRadius: radiusMap[variant],
        border: 'neutral',
        px: variant === 'search' ? '20' : '12',
        py: '8'
      })}
      {...props}
    />
  );
};
```

### 아바타

```tsx
const Avatar = ({ src, size = 'md', shape = 'circle' }) => {
  const sizeMap = {
    sm: '32',
    md: '48',
    lg: '64',
    xl: '96'
  };

  return (
    <div className={css({
      width: sizeMap[size],
      height: sizeMap[size],
      borderRadius: shape === 'circle' ? 'full' : 'lg',
      overflow: 'hidden'
    })}>
      <img src={src} alt="Avatar" />
    </div>
  );
};
```

## 모범 사례

### ✅ Do

- **일관된 사용**: 같은 레벨의 요소는 같은 둥글기
- **목적에 맞게**: 버튼은 md, 카드는 lg 등 용도별 구분
- **조화로운 중첩**: 중첩 시 적절한 간격 유지
- **브랜드 일관성**: 정해진 스케일 내에서 선택

### ❌ Don't

- **임의 값 사용**: 시스템 외부의 둥글기 값 지양
- **과도한 혼용**: 한 화면에 너무 다양한 둥글기
- **부자연스러운 조합**: 한 요소에 극단적으로 다른 값
- **overflow 누락**: 이미지 컨테이너에서 overflow: hidden 빼먹기

## 브라우저 호환성

모든 모던 브라우저에서 `border-radius` 속성을 완벽히 지원합니다.
- Chrome 4+
- Firefox 4+
- Safari 5+
- Edge (모든 버전)

구형 브라우저 대응이 필요한 경우:
```css
/* 벤더 프리픽스 (매우 오래된 브라우저용) */
-webkit-border-radius: 8px;
-moz-border-radius: 8px;
border-radius: 8px;
```
